## Projects Featured

**Employee Management System** — Full-stack Java app with a 10-table MySQL schema (3NF), role-based access, SHA-256 auth, and an automated test suite. Built as a team of 5 for SWE 3350.

**GSU Unofficial Campus Guide** — RAG system answering plain-language GSU student questions using ChromaDB, sentence-transformers, and Groq's LLaMA model. CodePath AI201 Project 1.

**FitFindr** — Multi-tool AI agent for searching secondhand clothing listings, outfit suggestions, and price comparison. CodePath AI201 Project 2.

**TakeMeter** — Fine-tuned DistilBERT classifier for indie music posts (`analysis` / `hot_take` / `reaction`). Includes zero-shot LLaMA baseline comparison and full error analysis. CodePath AI201 Project 3.

## Tech

- **HTML/CSS/JS** — no frameworks, no build step
- **Fonts** — Syne (display) + DM Sans (body) via Google Fonts
- **Animations** — CSS keyframes + IntersectionObserver scroll reveal + JS typing effect
- **Hosting** — GitHub Pages

---

## Running Locally

No dependencies or build tools needed.

```bash
git clone https://github.com/Sniggy26/Sniggy26.github.io.git
cd Sniggy26.github.io
open index.html        # macOS
# or just drag index.html into your browser
```
